# platform-behavior-baselines

各主要社群媒體平台的正常用戶行為基準線數據，用於異常檢測和操弄識別的參考標準。

## 通用行為基準

### 時間活動模式

#### 正常活動時段分佈

```
高峰時段:
- 07:00-09:00 (上班通勤時間) - 15-20%
- 12:00-14:00 (午餐時間) - 20-25%
- 18:00-22:00 (下班休閒時間) - 35-40%

低峰時段:
- 00:00-06:00 (深夜睡眠) - 5-8%
- 09:00-12:00 (工作時間) - 8-12%
- 14:00-18:00 (下午工作) - 10-15%
```

#### 週期性模式

```
週間模式:
- 週一至週五: 較規律的工作時間模式
- 週六週日: 時間分佈更平均，晚起晚睡

季節性變化:
- 夏季: 活動時間稍晚
- 冬季: 室內活動時間增加
- 假期: 時區混亂，不規律模式增加
```

### 互動行為模式

#### 正常回應時間

```
即時回應 (0-5分鐘): 20-30%
短期回應 (5分鐘-1小時): 40-50%
延遲回應 (1-24小時): 20-25%
長期回應 (>24小時): 5-10%
```

#### 內容長度分佈

```
短內容 (<50字): 60-70%
中等內容 (50-200字): 25-30%
長內容 (>200字): 5-15%
```

## Facebook/Meta 平台基準

### 用戶活動特徵

#### 貼文頻率

```
一般用戶:
- 每日貼文: 0-2則 (70%用戶)
- 重度用戶: 3-10則/日 (25%用戶)
- 極重度用戶: >10則/日 (5%用戶)

異常指標:
- >20則/日持續超過一週
- 每小時規律性貼文
- 24小時無間斷活動
```

#### 互動模式

```
按讚比例: 貼文的1-5%觸及率
留言比例: 按讚數的10-20%
分享比例: 按讚數的5-10%

異常模式:
- 按讚/留言比例 >10:1
- 分享率 >50%按讚數
- 所有互動在短時間內集中
```

#### 好友關係

```
正常範圍: 50-500好友
活躍互動: 10-50好友規律互動
密切關係: 5-15好友頻繁互動

異常指標:
- 好友數量 >2000 但互動少
- 新帳號快速累積大量好友
- 好友關係異常集中於特定群組
```

### 內容特徵

#### 主題多樣性

```
正常用戶:
- 3-7個主要興趣領域
- 生活、工作、娛樂內容混合
- 隨時間變化的興趣轉移

異常模式:
- 單一主題比例 >80%
- 突然的興趣主題轉變
- 缺乏個人生活內容
```

## Twitter/X 平台基準

### 用戶活動特徵

#### 推文頻率

```
一般用戶:
- 0-5推文/日 (60%用戶)
- 重度用戶: 6-30推文/日 (35%用戶)
- 極重度用戶: >30推文/日 (5%用戶)

異常指標:
- >100推文/日
- 每分鐘規律推文
- 推文時間間隔過於一致
```

#### 轉推與原創比例

```
正常比例:
- 原創推文: 30-60%
- 轉推: 25-50%
- 回覆: 15-30%

異常模式:
- 轉推比例 >90%
- 完全沒有回覆互動
- 只轉推特定來源
```

#### 追蹤關係

```
正常範圍:
- 追蹤者/追蹤中比例: 0.1-10
- 互動活躍度: 5-20%追蹤者

異常指標:
- 追蹤數量極不對稱
- 高追蹤數但零互動
- 快速追蹤/取消追蹤模式
```

### 語言使用模式

#### 語言多樣性

```
正常特徵:
- 詞彙豐富度適中
- 包含個人化表達
- 語言風格一致但有變化

異常特徵:
- 詞彙使用過於標準化
- 缺乏個人語言習慣
- 機械式的語言模式
```

## YouTube 平台基準

### 用戶互動特徵

#### 留言行為

```
正常模式:
- 留言長度: 10-100字
- 回覆延遲: 數分鐘到數小時
- 情緒表達自然變化

異常模式:
- 留言長度過於一致
- 即時回覆所有影片
- 情緒表達模板化
```

#### 訂閱模式

```
正常行為:
- 漸進式訂閱增長
- 興趣相關的頻道群組
- 取消訂閱率 <10%/月

異常行為:
- 大量快速訂閱
- 無關聯性的頻道組合
- 訂閱後無任何互動
```

## Instagram 平台基準

### 視覺內容特徵

#### 發文模式

```
正常模式:
- 1-3發文/週
- 個人生活與興趣混合
- 品質與隨意內容並存

異常模式:
- 過於規律的發文時間
- 內容風格過於一致
- 缺乏個人生活元素
```

#### 互動比率

```
正常比率:
- 按讚率: 追蹤者的2-10%
- 留言率: 按讚數的1-5%
- 限時動態觀看率: 20-60%

異常比率:
- 按讚率 <0.5% 或 >20%
- 留言全是表情符號
- 互動時間過於集中
```

## 檢測閾值設定

### 統計異常檢測

#### Z-score 閾值

```
輕微異常: |Z| > 2 (95%信心區間外)
中度異常: |Z| > 2.5 (99%信心區間外)
嚴重異常: |Z| > 3 (99.7%信心區間外)
```

#### 百分位數閾值

```
上限異常: >95th percentile
極端異常: >99th percentile
下限異常: <5th percentile
```

### 行為模式評分

#### 協調性評分 (0-10)

```
0-2: 正常個人行為
3-4: 輕微協調跡象
5-6: 可疑協調行為
7-8: 明顯協調操作
9-10: 高度自動化操作
```

#### 真實性評分 (0-10)

```
0-2: 明顯機器人行為
3-4: 可疑自動化帳號
5-6: 真實性存疑
7-8: 可能真實帳號
9-10: 高可信度真實帳號
```

## 文化和地域差異

### 時區考量

```
需要調整的因素:
- 不同時區的活躍時間
- 文化習慣的作息差異
- 工作日/假日的變化
```

### 語言模式差異

```
考慮因素:
- 語言特有的表達方式
- 文化背景的影響
- 社會議題的敏感度
```

### 平台使用習慣

```
地域差異:
- 平台普及率差異
- 使用目的的文化差異
- 互動禮儀的不同
```
